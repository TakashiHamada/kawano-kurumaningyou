# 川野車人形 練習用音声プレーヤー

川野車人形（かわのくるまにんぎょう）の語りを練習するための、Web ベースの音声プレーヤーアプリケーションです。

## 川野車人形とは

川野車人形は、日本の伝統的な人形芝居の一つです。本アプリでは、熟練の演者による貴重な録音を聴きながら語りの練習を行うことができます。

## 使い方

1. ブラウザで `index.html` を開きます
2. ドロップダウンメニューから聴きたい音声を選択します
3. 「再生」ボタンで再生、「5秒戻す」ボタンで巻き戻しができます

## 主な機能

- 音声の再生・一時停止
- 5秒巻き戻し
- 演目・話者・収録日の表示
- 関連画像（台本など）の閲覧 ※画像が登録されている音声のみ
- スマートフォン対応（縦向き表示）

## 関連画像（台本）の表示について

音声に関連する画像（台本・詞章など）を登録すると、その音声を選んだとき
だけ「📖 台本を見る」ボタンが表示されます。ボタンを押すと画像が全画面で
開き、複数枚ある場合は「前 / 次」で切り替えられます。画像をタップすると
拡大・縮小、画像の外側をタップまたは「×」で閉じられます。

## 新しい音声ファイルの追加方法

1. MP3 形式の音声ファイルを `index.html` と同じフォルダに配置します
2. `audio_metadata.csv` をテキストエディタで開き、末尾に新しい行を追加します

```
ファイル名,演目名,話者名,収録年月日,画像ファイル名
```

最後の「画像ファイル名」は任意です。関連画像がない場合は空のままにします。
複数の画像を登録するときは、ファイル名を縦棒（`|`）で区切ります。

**例:**

```csv
filename,title,speaker,recorded_date,images
kowakare_furuya.mp3,子別れ,古谷要平,昭和51年7月11日,
watashiba_sugita.mp3,保存版「渡し場」三味線なし,杉田芳春,平成8年頃,
togo-kowakare_haruji.mp3,当吾 子別れ,島崎春治,平成23年3月,
practice_sugita_0.mp3,練習法1,杉田芳春,平成6年11月12日,practice_0.jpg|practice_1.jpg   ← 画像2枚を登録
newfile.mp3,新しい演目,演者名,令和○年○月,    ← この行を追加（画像なし）
```

**注意事項:**

- 音声ファイルは MP3 形式のみ対応しています
- CSV の各項目はカンマ（`,`）で区切ってください
- 画像ファイルは `index.html` と同じフォルダに置き、複数枚は縦棒（`|`）で区切ります
- ファイル名に日本語やスペースは使わず、半角英数字とハイフン・アンダースコアを推奨します
